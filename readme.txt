# 🎧 SoulSonickRender

SoulSonickRender is a mood-based music exploration app powered by the Spotify API. Select how you're feeling, and it’ll serve you a fresh set of tracks. This project is equal parts fun, functional, and full-stack.

---

## 🔥 Features

- 🎵 Get Spotify tracks based on your current mood (Happy, Sad, Chill, Energetic)
- ⚡ Secure backend for API token handling (no exposed secrets)
- ✨ Smooth UI with hover animations, toast alerts, and a loading spinner
- 🖼️ Album covers, track previews, and one-click Spotify links
- 📱 Fully responsive — works great on desktop and mobile
